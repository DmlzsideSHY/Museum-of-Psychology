# 记忆地层 · 心理博物馆/Museum of Psychology

谨以此项目，献给曾经失意的我，以及那短暂却深刻的高中三年。


那些破碎的、温暖的、沉默的、闪光的时刻，都被我一块一块地埋进这里，成了化石。

如果你也正在经历一段难熬的日子，希望这个博物馆能陪你坐一会儿。

---

## 项目简介

记忆地层 · 心理博物馆是一个用于记录、可视化和探索个人记忆与情绪的个人知识管理工具。它将你的记忆像地质地层一样分层展示，并以"化石"为隐喻，帮助你在时间与情绪之间建立连接。

你可以：

- 埋藏记忆，带日期、情绪、文字、图片
- 挖掘记忆，随机抽取，像考古一样
- 建立记忆之间的关联，形成"记忆基因"网络
- 用时间线串联人生片段
- 在星云、年轮、心理地图中看见自己的情绪轮廓

---

## 功能列表

### 记忆化石系统

记录记忆的标题、详细描述、情绪标签、具体日期或模糊时期，如"童年"、"大学"。

### 四种可视化视图

- 地层视图：按时间远近分层展示记忆，地表、近代、深层
- 铺排视图：列表式浏览，支持按内容类型筛选，全部、仅图片、仅文字
- 照片墙视图：以图片瀑布流形式展示带有配图的记忆
- 年轮视图：3D 年轮式空间布局，每一圈都是一个记忆

### 记忆基因关联

- 支持在记忆之间建立双向关联，如"导致"、"治愈"、"转折"，形成记忆网络
- 自动识别"连接体"，互相连通的记忆群组，一键生成时间线

### 情绪统计与星云

- 情绪标签频次统计表格
- 3D 星云视图，按情绪聚类展示记忆，点击可查看详情

### 心理地图

- 基于关联网络生成节点图
- 节点支持拖拽、缩放，显示入度/出度端口
- 支持按时间线上色，导出为 SVG

### 时间线与分类筐

- 创建自定义时间线，如"高中三年"、"那个夏天"
- 支持拖拽排序、自由组合化石
- 可将时间线归类到分类筐中

### 情绪急救 · 抱抱便签

- 随机抽取一条温暖记忆，附上鼓励语，提供情绪支持
- 当你需要一点温暖时，点一下"抱抱"按钮

### 数据管理

- 支持导出/导入数据，JSON 格式，含图片 base64
- 支持纯文本导出、Markdown 导出
- 支持打印和生成 PDF 报告

## 技术栈

- 前端框架：原生 JavaScript + HTML + CSS，无框架依赖
- 数据存储：IndexedDB + localStorage，双备份

可视化库：

- Three.js：星云 / 年轮 3D 视图
- D3.js：心理地图节点图
- html2canvas + html2pdf：截图与 PDF 导出

## 使用说明

### 初次使用

页面会自动生成示例数据，方便你快速体验所有功能。你可以保留它们作为参考，也可以全部删掉，从头开始。

### 埋藏记忆

点击底部"埋藏"按钮：

1. 填写具体日期或选择大致时期，如"高一"、"大学"
2. 选择或添加情绪标签
3. 填写标题和详细描述
4. 可选：上传配图，支持多张
5. 可选：建立与其他化石的关联，双向

### 挖掘记忆

点击"挖掘"按钮，系统会基于当前季节推荐一条记忆，随机出土。支持查看和编辑。

### 切换视图

顶部有四种视图模式：

- 地层：按时间分层
- 铺排：列表浏览
- 照片墙：图片瀑布流
- 年轮：3D 环形布局

### 时间线

点击底部"时间线"按钮：

1. 创建时间线，如"高中三年"
2. 从化石列表中勾选加入
3. 支持拖拽排序
4. 可在"时间线详情"中查看、排序、高亮显示在地层中

### 统计与地图

在"统计"页面：

- 查看情绪分布表格或 3D 星云
- 查看"心理地图"，基于关联网络生成的节点图
- 可导出地图为 SVG

### 抱抱

点击树下的"抱抱"按钮，会随机抽取一条温暖的记忆，并附上一句鼓励语。

---

## 数据备份与迁移

- 导出数据：设置 -> 导出数据，完整 JSON，含图片 base64
- 仅文本导出：设置 -> 仅文本导出，不含图片，文件更小
- Markdown 导出：设置 -> 导出 MD，支持自定义排序和内容范围
- 导入数据：设置 -> 导入数据，选择之前导出的 JSON 文件即可恢复

---

## 自定义扩展建议

- 修改 defaultSettings.quoteLibrary 中的欢迎语列表，在 JS 中
- 在 ALLOWED_IMAGE_TYPES 中扩展支持的图片格式
- 调整 emotion-tags 中的预设情绪标签，在 HTML 中
